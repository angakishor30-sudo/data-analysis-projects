📊 Retail Sales Analytics Dashboard (Excel)
📌 Project Overview

This project is an interactive Retail Sales Analytics Dashboard built entirely in Microsoft Excel. The dashboard provides a comprehensive overview of sales performance across different products, locations, and time periods, enabling users to analyze key business metrics and make data-driven decisions.

The dashboard utilizes Excel's powerful data analysis features, including Pivot Tables, Pivot Charts, Slicers, Timelines, and KPI Cards, to transform raw sales data into meaningful insights.

🎯 Objectives
Monitor overall sales performance.
Analyze product-wise sales contribution.
Compare sales across different locations.
Track monthly sales trends.
Enable dynamic filtering through slicers and timelines.
Provide a user-friendly and interactive reporting experience.
📂 Dataset Information

The dataset contains retail sales transactions with the following fields:

Column	Description
Date	Transaction Date
Sales Person	Sales Representative
Product	Product Category
Place	Sales Location
Price	Product Price
Units	Quantity Sold
Amount	Total Sales Amount

Visualizations
1. Sales by Products

Displays the percentage contribution of each product category to overall sales.

2. Sales by Months

3. Sales by Places

Compares sales performance across different locations.
